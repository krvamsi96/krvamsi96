### 🌟 Featured Projects

Here are some of my top open-source projects focusing on AI, Machine Learning, and Software Engineering:

#### 🤖 Generative AI & Agents
*   **[medical-appointment-agen](https://github.com/krvamsi96/medical-appointment-agen)**
    *   *Description:* An automated, intelligent medical agent designed to parse requirements and execute appointments seamlessly using LLMs.
    *   *Tech Stack:* Python, GenAI, LangChain, FastAPI.
*   **[langgraph](https://github.com/krvamsi96/langgraph)**
    *   *Description:* Production-ready LangGraph agent workflows demonstrating multi-agent orchestration, state management, and memory systems.
    *   *Tech Stack:* Python, LangGraph, Jupyter Notebook.
*   **[Web-Content-Q-A-Tool](https://github.com/krvamsi96/Web-Content-Q-A-Tool)**
    *   *Description:* A Streamlit dashboard that processes raw web content from URLs and handles document QA using retrieval methods via Groq API.
    *   *Tech Stack:* Python, Streamlit, Groq API, LangChain.
*   **[LangChain-Document-Processing-and-QA](https://github.com/krvamsi96/LangChain-Document-Processing-and-QA)**
    *   *Description:* Retrieval-Augmented Generation (RAG) pipeline built using LangChain and OpenAI to process unstructured documents, construct vector store embeddings, and perform intelligent QA.
    *   *Tech Stack:* Python, LangChain, OpenAI API, Vector DB.

#### 👁️ Computer Vision & Predictive ML
*   **[YOLO-object-detection](https://github.com/krvamsi96/YOLO-object-detection)**
    *   *Description:* Implementation of YOLO algorithms for high-speed, real-time object detection and labeling.
    *   *Tech Stack:* OpenCV, YOLO, Python.
*   **[Genre_Prediction_from_Movie_Posters](https://github.com/krvamsi96/Genre_Prediction_from_Movie_Posters)**
    *   *Description:* Multi-label classification models predicting film genres using Deep Learning directly from movie poster visual art.
    *   *Tech Stack:* PyTorch, TensorFlow, Computer Vision, Jupyter Notebook.
*   **[face-emotion-recognition-python](https://github.com/krvamsi96/face-emotion-recognition-python)**
    *   *Description:* Real-time facial emotion analyzer detecting sentiments using neural networks from video streams.
    *   *Tech Stack:* Python, OpenCV, Deep Learning.

#### 🔠 NLP & Text Analytics
*   **[Preprocessing-Text-Python-Package](https://github.com/krvamsi96/Preprocessing-Text-Python-Package)**
    *   *Description:* A custom Python package designed to streamline and automate text preprocessing tasks in NLP pipelines, supporting tokenization, lemmatization, and stop-word removal.
    *   *Tech Stack:* Python, NLP, Regular Expressions, NLTK/SpaCy.
*   **[Extract-Stock-Sentiment-From-News-Headlines](https://github.com/krvamsi96/Extract-Stock-Sentiment-From-News-Headlines)**
    *   *Description:* A financial sentiment analysis tool that web scrapes live stock headlines and classifies market sentiment using NLP sentiment analyzers to forecast trends.
    *   *Tech Stack:* Python, Beautiful Soup, VADER Sentiment, Pandas, HTML.
